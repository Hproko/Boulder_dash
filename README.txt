Esta eh uma implementacao em C do jogo Boulder_dash para a disciplina de Programacao 2. 


O jogo consiste em coletar diamantes e abrir o portal de fuga do personagem, o jogador que completar esta tarefa em menos tempo vai ganhar mais pontos. 

Para ativar o cheat code do jogo, basta digitar "explode". Use com moderacao. Esse cheat foi pensado para ser usado nos momentos em que o jogador ficou preso pelas 
pedras.


Bom jogo!
